# Journal_file.py

Содержит класс Journal с атрибутом _journal - словарь, где будут хранитьсяч студенты и оценки

### Функции:
  - add_student (добавляет студента с оценкой в _journal)
  - update_mark (приверяет есть ли студентом с заданным именем и если есть - изменяет его оценки на заданный список)
  - get_mark (возвращает список оценок студента по имени)
  - get_all (выводит всех студентов и их оценки в строку)
